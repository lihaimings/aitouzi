# A 股与 ETF · AI 投资开源调研计划

范围：A 股与 ETF（宽基/行业/主题），纸交易为主；LLM 作为资讯/情绪与研究自动化辅助。

交付（v0，24–48h）：
- 开源清单（10–20 项）：用途、星标、维护度、License、优缺点、适用场景、是否便于接入自有 LLM API
- 推荐组合：数据→回测→组合→纸盘→报告的最小链路
- 示例 Notebook 设计稿（ETF 轮动 + 情绪开关）

评审要点：
- 数据可得性（A 股/ETF）、约束（T+1、涨跌停、停牌）、交易成本建模
- 回测严谨性（walk-forward、参数冻结）、绩效指标（超额/回撤/卡玛）
- LLM 价值边界（不用作唯一信号）、API 成本与速率
