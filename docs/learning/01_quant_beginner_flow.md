# 量化投资小白入门：流程、重点与学习路径

你可以把量化投资理解成一条流水线：

1. 提问题：我要在什么市场、用什么规则赚钱？
2. 找数据：拉取历史行情与辅助因子
3. 写策略：把规则写成代码（可重复执行）
4. 跑回测：看历史上表现如何
5. 做风控：限制回撤、仓位、换手
6. 纸交易：不动真金白银先模拟
7. 复盘优化：看问题、改参数、再验证
8. 小资金实盘：仅在稳定后逐步上线

## 一、量化投资标准流程（你先记住这7步）

## 1) 明确目标
- 不是“每月暴赚”，而是“长期稳定、可复现、可解释”。
- 先定义约束：最大回撤、仓位上限、交易频率。

## 2) 数据准备
- 价格数据：开高低收、成交量、成交额。
- 辅助数据：新闻、情绪、宏观、行业等（可选）。
- 核心原则：数据必须可追溯、可重复下载。

## 3) 策略设计
- 把想法变成明确规则：
  - 何时买？
  - 买什么？
  - 买多少？
  - 何时卖？
- 避免“拍脑袋规则”，每条规则都要能写成代码。

## 4) 回测验证
- 看收益，也看风险：
  - 年化收益、夏普、最大回撤、胜率、换手率。
- 一定做滚动验证（Walk-Forward），避免只在某一段历史表现好。

## 5) 风险控制
- 设置硬阈值（建议）：
  - 最大单日回撤
  - 最大总回撤
  - 单标的最大仓位
- 量化策略先活下来，再谈收益。

## 6) 纸交易
- 连续跑几个月到一年。
- 看策略在牛市、熊市、震荡是否都能接受。

## 7) 复盘 + 小步优化
- 每周/月固定复盘。
- 先改一个变量，再观察，不要同时改很多参数。

## 二、小白最容易踩的坑

- 只看收益，不看回撤。
- 过度调参，结果过拟合。
- 数据断点/脏数据没处理，导致回测“好看但假”。
- 频繁换策略，没有稳定执行周期。

## 三、你现阶段的学习重点（按优先级）

1. 先学会读懂回测报告（收益、回撤、夏普）
2. 学会看风控阈值是否触发
3. 学会按“审批流程”修改参数，而不是直接改策略代码
4. 学会做周/月复盘，建立策略日志

## 四、学习节奏建议（4周）

- 第1周：熟悉项目运行链路，读日报和月报。
- 第2周：理解策略参数含义，学会审批模板。
- 第3周：看Walk-Forward和市场阶段复盘。
- 第4周：尝试一次“参数建议 -> 人工审批 -> 复测”的完整闭环。

如果你始终坚持“先风险、后收益；先模拟、后实盘”，你就已经在正确道路上。
